Dorchester Guide v17.0

A mobile-first redesign built on v16.1. It adds a new Home screen, five-tab navigation, category-based Explore page, unified app search, compact/expanded cards, full place views, improved dark mode and a cleaner daily-use flow while preserving existing Google Places, maps, notes, saved/visited data, galleries and itineraries.

Upload all five files to the repository root, replacing the existing files. Wait for GitHub Pages to redeploy, then reload the site. If an older interface remains, close the browser tab and reopen it so the v17 service worker can replace the previous cache.
