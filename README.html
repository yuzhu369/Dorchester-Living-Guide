Google Cloud

Use one browser key restricted to https://yuzhu369.github.io/*. Restrict the key to Maps JavaScript API and Places API. This v15 build uses the browser-side Maps JavaScript Places Library and no longer calls Place.searchByText / the Places API (New) SearchText endpoint.

After uploading

Wait for GitHub Pages to redeploy, then hard-refresh the page. On macOS use Command + Shift + R; on Windows use Ctrl + Shift + R. Paste your browser-restricted key into “Add exact place photos”.
